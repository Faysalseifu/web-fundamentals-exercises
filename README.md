## 🌟 Overview

This repository documents my journey of mastering web fundamentals. The project is strategically split between **Frontend** and **Backend** to provide a clear separation of concerns as the learning complexity scales from simple HTML components to server-side environments and databases.

## 📂 Folder Structure

The workspace is thoughtfully organized into sequential learning modules:

```text
web-fundamentals-exercises/
├── 🎨 frontend/
│   ├── 01-html/         # Basic tags, layouts, and forms
│   ├── 02-css/          # Layout techniques, selectors, Flexbox, & Media Queries
│   ├── 03-bootstrap/    # Component framework and responsive grid systems
│   └── 04-javascript/   # DOM manipulation, variables, and logic (Coming Soon)
│
├── ⚙️ backend/          (Coming Soon)
│   ├── 01-nodejs/       # JavaScript runtime exercises
│   ├── 02-express/      # Server creation and API endpoints
│   └── 03-databases/    # SQL / NoSQL database integration
│
└── README.md
```

## 🛠️ Technologies Covered

The exercises currently feature detailed examples of:
- **HTML5:** Semantic structure, complex forms, inline vs. block elements.
- **CSS3:** The Box model, classic positioning, modern Flexbox layouts, and Mobile-First Media Queries for responsive design.
- **Bootstrap 5:** Rapid UI prototyping, grid responsiveness, and integrated components (Cards, Alerts, Tables, and Forms).

## 🚀 Getting Started

To explore the exercises:
1. Clone this repository to your local machine.
2. Navigate through the nested folders starting from `frontend/01-html/`.
3. Open any `.html` file in a modern web browser to view the interactive examples. 
4. Read the extensively commented HTML and CSS files to understand the mechanics behind each implementation.
